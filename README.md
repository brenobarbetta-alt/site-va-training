# Site VA Training

Site oficial do VA Training — plataforma de treinamento para assistentes virtuais.

## Sobre

O VA Training oferece cursos e materiais para capacitação de assistentes virtuais profissionais.

## Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/brenobarbetta-alt/site-va-training.git

# Abra o index.html no seu navegador
```
